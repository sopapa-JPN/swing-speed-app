# スイングスピード測定アプリ

スマホ動画からバット先端を追跡し、スイング速度を km/h で表示するブラウザアプリです。
動画はサーバーへ送信せず、端末内で解析します。

## GitHub Pagesで公開

1. このフォルダ内のファイルを、GitHubリポジトリの一番上にアップロードします。
2. GitHubの「設定」→「Pages」を開きます。
3. 公開元を「Deploy from a branch」にします。
4. ブランチを「main」、フォルダーを「/(root)」にして保存します。
5. 数分後に表示されるURLを開きます。

## 精度を上げる撮影方法

- バッターの真横から撮る
- カメラを固定する
- バット先端に明るい単色テープを巻く
- 背景とテープの色を変える
- バット全体を画面に入れる
- 撮影モードを正しく選ぶ

## 注意

スマホ動画による推定値です。公式計測器と完全に同じ値を保証するものではありません。同じ撮影条件で成長を比較する用途に向いています。
